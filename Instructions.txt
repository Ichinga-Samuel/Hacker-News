
To load data from HackerNews api
run the following management commands

python manage.py top_stories --comments  
This loads the top stories from the api and recursively load their comments and replies to the comments


python manage.py top_stories. 
Quickly load the top stories without comments

Other Commands
can be called with or without the comments flag
top_jobs, ask_stories, show_stories load_latest


API Endpoints
/api/stories/
/api/jobs
/api/create_story
/api/edit_story/id
/api/create_job/
/api/edit_job/
